# Data & Business Services Platform - Frontend

This is the frontend application for the Data & Business Services Platform built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

### Prerequisites
- Node.js 18+ and npm/yarn/pnpm
- Backend server running (see backend README)

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
# or
yarn install
# or
pnpm install
